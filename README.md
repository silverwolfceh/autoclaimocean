# How To use
- ```git clone https://github.com/silverwolfceh/autoclaimocean.git```
- ```cd autoclaimocean```
- ```npm i```
- ```edit data.json.ex to fill your mnemonic```
- ```Rename data.json.ex to data.json```
- ```node index.js```
